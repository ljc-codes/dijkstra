  1) File Name: Dijkstra.java

    Compilation:
      javac Dijkstra.java

  _____________________________________________________________________

  2) File Name: Vertex.java

    Description: Vertex class for Dijkstra.java

    Compilation:
      javac Vertex.java

  _____________________________________________________________________

  3) File Name: Edge.java

    Description: Edge class for Dijkstra.java

    Compilation:
      javac Edge.java
  _____________________________________________________________________

  2) File Name: Display.java

    Description: provided GUI for displaying graph and applying Dijkstra
                  methods.

    Compilation and Run Instructions:
      javac Display.java
      java Display
  _____________________________________________________________________

  3) Additional Data files:
  citypairs.txt - edges for vertices
  cityxy.txt - x,y coordinates of cities.
